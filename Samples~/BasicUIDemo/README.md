# Basic UI Demo

This folder is a placeholder for the bundled sample.

**Before publishing to the Asset Store**, replace this README with:
- `ExampleScene.unity` — a scene with a Canvas containing 1-3 example buttons
- A `SampleButtons/` folder with the example `.prefab` files referenced by the scene
- Any sprites the example prefabs need (PNG + their `.meta` files)

To create the sample:
1. In your Unity project, open `Tools > Gami Prefab Importer`.
2. Import any test ZIP from arteditor.art.
3. Drag the resulting prefabs into a new scene.
4. Save the scene as `ExampleScene.unity` here.
5. Move the prefab + sprite assets into `SampleButtons/`.

Once the sample is in place, **delete this README.md**.
